# Global Attributes

Global attributes are key-value pairs that are automatically added to all telemetry data (spans and logs) generated by the AWS OpenTelemetry Swift SDK. Unlike resource attributes, these are mutable at runtime and are appended as regular attributes.

## Usage

```swift
import AwsOpenTelemetryCore

// Get the global attributes manager
let manager = AwsGlobalAttributesProvider.getInstance()

// Set global attributes
manager.setAttribute(key: "service.version", value: AttributeValue.string("1.0.0"))
manager.setAttribute(key: "Foo", value: AttributeValue.string("bar"))

// All subsequent spans and logs will include these attributes
```

## Example Output

When global attributes are set, they appear in all telemetry data. For example, a session log record will include both session-specific attributes and your global attributes:

```jsonc
{
  "body": { "stringValue": "session.start" },
  "attributes": [
    { "key": "user.id", "value": { "stringValue": "1CBF1932-6111-4C87-89A3-8CD7B971187B" } },
    { "key": "session.id", "value": { "stringValue": "B42804E3-4959-4B7B-A7F2-8D72252A978A" } },
    { "key": "session.start_time", "value": { "doubleValue": 1758577993338341000 } },

    // global attributes
    { "key": "Foo", "value": { "stringValue": "bar" } },
    { "key": "service.version", "value": { "stringValue": "1.0.0" } }
  ]
}
```
